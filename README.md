# Does it have wifi?

## Development

Requires **Node 0.12+**.

Install:

```bash
$ npm install -g gulp
$ npm install
```

Run server with auto-reload (without minification):

```bash
$ gulp
```

To force minification of assets (build takes a bit longer) include the `--minified` flag:

```bash
$ gulp --minified
```
